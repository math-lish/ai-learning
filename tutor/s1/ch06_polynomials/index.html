1. 數學語言

認識多項式的相關術語：

3x² + 5x - 2
↑ ↑ ↑
項 項 項

• 項 (Term)：由字母和數字組成的部分

• 係數 (Coefficient)：項中的數字部分

• 常數項 (Constant)：不含字母的項

• 項數 (Number of Terms)：多項式包含的項數

• 次數 (Degree)：最高次冪的冪數

2. 同類項

同類項：具有相同字母部分的項

例：判斷同類項

• 3x² 和 5x² → 同類項（都係 x²）

• 4x 和 7x → 同類項（都係 x）

• 3x² 和 3x → 不同類項（次數唔同）

合併同類項：將同類項的係數相加減

3x + 5x = 8x
7x² - 2x² = 5x²

3. 單項式乘除法

單項式乘法：將數字相乘，相同字母的指數相加

3x² × 4x³ = 12x⁵ (3×4=12, x²⁺³=x⁵)

單項式除法：將數字相除，相同字母的指數相減

12x⁵ ÷ 3x² = 4x³ (12÷3=4, x⁵⁻²=x³)

4. 多項式乘法

📐 日字法（一項 × 兩項）

當一項式乘兩項式時使用

2x × (x + 3)
= 2x·x + 2x·3
= 2x² + 6x
📐 田字法（兩項 × 兩項）

當兩項式乘兩項式時使用田字格

(x + 2)(x + 3)
= x·x + x·3 + 2·x + 2·3
= x² + 3x + 2x + 6
= x² + 5x + 6

5. 多項式減法

去括號法則：括號前是減號，去括號要變號

(3x² + 5x) - (x² + 2x)
= 3x² + 5x - x² - 2x
= 2x² + 3x

📝 測驗題目

Question 1 (MC)

化簡：5x² + 3x - 2x² + 4x

A. 3x² + 7x    B. 7x² + 7x    C. 3x² + 7x    D. 7x² + x

答案：A. 3x² + 7x
解說：5x² - 2x² = 3x²，3x + 4x = 7x

Question 2 (MC)

計算：6x⁴ ÷ 2x²

A. 3x²    B. 4x²    C. 3x⁶    D. 4x⁶

答案：A. 3x²
解說：6÷2=3，指數相減：4-2=2

Question 3 (MC)

計算：(x - 3)(x + 4)

A. x² + x - 12    B. x² - x - 12    C. x² + 7x - 12    D. x² - 7x + 12

答案：B. x² + x - 12
解說：田字法：x·x + x·4 - 3·x - 3·4 = x² + 4x - 3x - 12 = x² + x - 12

Question 4 (Long Question)

化簡以下多項式：

(a) 4x(x - 3) + 2x(x + 5)

(b) (x + 1)²

答案：
(a) = 4x² - 12x + 2x² + 10x = 6x² - 2x
(b) = (x+1)(x+1) = x² + 2x + 1
