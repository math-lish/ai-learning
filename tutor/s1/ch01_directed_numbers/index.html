📖 教學講解

咩係正數同負數？

正數 (Positive Numbers)：大過0既數，前面有「+」號或者唔寫符號。例如：+5, 3, 7.2

負數 (Negative Numbers)：細過0既數，前面有「-」號。例如：-5, -3, -7.2

0 既唔係正數亦都唔係負數，佢係正負數既分界線！

重要規則

1. 相反數 (Opposite Numbers)

+5 同 -5 係相反數，佢地既絕對值相同，但符號相反。

2. 絕對值 (Absolute Value)

|+5| = 5 |-5| = 5 （絕對值永遠係正數或零）

加法運算

📝 例題 1：同號相加

(+3) + (+5) = +8

(-3) + (-5) = -8

同號就保留符號，把絕對值相加

📝 例題 2：異號相加

(+5) + (-3) = +2 （|5| > |3|，答案取較大絕對值既符號）

(+3) + (-5) = -2 （|5| > |3|，答案取較大絕對值既符號）

減法運算

📝 例題 3：減負數

5 - (-3) = 5 + 3 = 8

減去一個負數等於加上佢既相反數！
